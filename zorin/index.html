Zorin OS 18

An Ubuntu 24.04 base with a Zorin-themed GNOME desktop, streamed securely through noVNC in your browser.

Open Desktop
Browser access: enter the VNC_PASSWORD you supplied when starting the container.

RDP app access: on RunPod, expose TCP 3389, then use the Direct TCP public IP and assigned external port with username zorin and the same VNC_PASSWORD.
